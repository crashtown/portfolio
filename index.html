Projects

NodeJS LDAP SOCKS5


NostConnect is a project I created back in 2017, this service lowers in-game latency for World of Warcraft players in Australia. The service is split over two servers, one running NodeJS and the other a SOCKS5 proxy, user authentication between them is provided by LDAP and the Paypal API is implemented to collect monthly revenue.
